# 一键部署教程（5分钟搞定）

## 方案：Vercel 部署（推荐）

Vercel 是免费的静态网站托管平台，部署后你会得到一个永久可用的链接。

---

## 步骤 1：准备代码（1分钟）

### 方法 A：直接下载 ZIP（最简单）
1. 在当前页面，点击菜单找到「下载代码」或「导出项目」
2. 下载后解压到桌面

### 方法 B：用 GitHub（如果你有账号）
1. 访问 https://github.com/new
2. 仓库名称填 `health-checkin`
3. 点击 "Create repository"
4. 上传代码到该仓库

---

## 步骤 2：注册 Vercel（1分钟）

1. 访问 https://vercel.com
2. 点击 "Sign Up"
3. 选择 "Continue with GitHub"（用 GitHub 账号登录）
4. 完成注册

---

## 步骤 3：部署项目（2分钟）

1. 登录 Vercel 后，点击 "Add New Project"
2. 选择你刚才创建的 GitHub 仓库（或上传代码的仓库）
3. 在配置页面，填写以下信息：

   | 配置项 | 填写内容 |
   |--------|----------|
   | Framework Preset | `Other` |
   | Build Command | `npm run build:client` |
   | Output Directory | `dist/client` |

4. 点击 "Deploy"
5. 等待 1-2 分钟，部署完成！

---

## 步骤 4：获取你的永久链接（30秒）

部署成功后，Vercel 会给你一个链接，例如：

```
https://health-checkin-yourname.vercel.app
```

**这就是你的永久 App 链接！** 可以：
- 收藏到微信
- 添加到手机桌面
- 分享给朋友

---

## 步骤 5：迁移数据（1分钟）

**重要**：新链接的数据和飞书链接是分开的！

### 备份原数据（在飞书链接中操作）：
1. 打开飞书中的健康打卡
2. 滚动到底部「数据管理」
3. 点击「备份数据」
4. 点击「复制到剪贴板」
5. 发送到微信收藏或备忘录保存

### 恢复到新链接：
1. 打开 Vercel 部署的新链接
2. 滚动到底部「数据管理」
3. 点击「恢复数据」
4. 粘贴之前复制的备份代码
5. 点击「恢复」

**完成！** 所有数据都迁移过来了。

---

## 添加到手机桌面（可选）

### iPhone：
1. 用 Safari 打开你的 Vercel 链接
2. 点击底部分享按钮 ⬆️
3. 选择「添加到主屏幕」
4. 给 App 起个名字（如「健康打卡」）
5. 点击「添加」

### Android：
1. 用 Chrome 打开链接
2. 点击右上角「⋯」菜单
3. 选择「添加到主屏幕」或「安装应用」

---

## 常见问题

**Q：部署后数据会丢失吗？**
A：不会，数据存在你的手机浏览器里。但飞书链接和新链接是独立的存储空间，需要按步骤 5 迁移。

**Q：Vercel 免费吗？**
A：完全免费，个人使用无限制。

**Q：以后换手机怎么办？**
A：先备份数据 → 换新手机 → 访问同一个 Vercel 链接 → 恢复数据。

**Q：可以绑定自己的域名吗？**
A：可以，在 Vercel 项目设置中添加自定义域名。
